# Resource Hub

## Tech Stack

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Shadcn UI**
- **Firebase** (Firestore)
- **NextAuth** (Authentication)
- **Nodemailer** & **React Email** (Email notifications)

## Features

- **Curated Resource Section:**  
  Dedicated area for students to discover and access high-quality, relevant resources.

- **Student Resource Contribution:**  
  Students can submit and share valuable resources with the community.

- **Admin Panel for Quality Control:**  
  Admins can verify, edit, or remove resources to maintain content quality.

## Highlightss

- Modern UI with responsive design.
- Authentication and user management via NextAuth.
- Features an admin panel for resource verification, editing, and quality control.
- Firebase Firestore for real-time data storage.
- Modular, scalable codebase with best practices.
- Email notifications for resource submissions and verifications using Nodemailer and React Email.
